# Tip-Calculator
# 💰 Tip Calculator App

A simple and user-friendly Android application built using **Kotlin** that helps users quickly calculate tips based on bill amount and selected percentage.

---

## 🚀 Features

* 📌 Real-time tip calculation
* 📌 Clean and responsive user interface
* 📌 Customizable tip percentage
* 📌 Input validation for better user experience
* 📌 Fast and lightweight performance

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI:** XML
* **IDE:** Android Studio

---


## ⚙️ How It Works

1. Enter the total bill amount
2. Select or enter the tip percentage
3. App instantly calculates the tip and total amount

---

## 🧠 Key Learnings

* Understanding of Android Activity Lifecycle
* Handling user input and validations
* Designing responsive UI using XML
* Implementing real-time calculations

---

## 📂 Project Structure

* `MainActivity.kt` – Handles core logic and UI interaction
* `activity_main.xml` – Defines UI layout
* `utils/` (if any) – Helper functions
